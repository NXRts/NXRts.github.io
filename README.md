# Arrofi Portfolio

A modern, responsive, and minimalist single-page portfolio website built with HTML and CSS.

![Portfolio Preview](img/akebi.jpg)

## üöÄ Features

- **Single Page Application (SPA) Style**: High-performance single-page layout with smooth hash-link navigation.
- **Sticky Glassmorphism Navbar**: A modern, semi-transparent navigation bar that stays pinned to the top with a blur effect.
- **Pure CSS Mobile Menu**: A fully functional hamburger menu for mobile devices implemented without heavy JavaScript.
- **Responsive Design**: Optimized for all screen sizes, from large desktops to small mobile devices (using `svh` and `clamp`).
- **Smooth Scroll**: Automatic smooth scrolling to sections when navigation links are clicked.
- **Integrated Social Media**: Quick access to Facebook, Instagram, GitHub, Twitter, and YouTube.
- **Contact Form**: A clean, ready-to-use contact form section.
